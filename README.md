# 반려동물 관리 
## 프로젝트 목표
- 반려동물을 키우면서 아프거나 불편한 점을 바로 알아차리지 못해 병원을 늦게 가는 경우가 많다. 이로 인해 반려동물의 건강 문제와 함께 치료비 부담이 커질 수 있어 이를 예방하고 관리할 수 있는 웹 서비스를 만들게 되었다.

## 사용 기술
- Javascript
- React native
- 몽고DB

## 구현 내용
- 캘린더 기능
- AI 상담 챗봇
- 애완동물 추가 기능
- 체크 리스트

## 느낀 점
- React를 처음 사용해 보는 프로젝트라 어려움이 있었지만 강의 영상을 참고하고 친구들과 의견을 나누며 코드를 작성하고 기능을 구현해 나가는 과정이 매우 뿌듯했다.
